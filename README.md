# Cetinje Bible / Цетиньская Библия  
## “Христианам третьего тысячелетия” (v2.2)

This repository hosts the first public release of the philosophical and theological work **“Христианам третьего тысячелетия”** (“To Christians of the Third Millennium”), as well as related TLS/TLS7 materials used in the broader *Cetinje Bible* project.

## Contents
- **Main text** (Russian + translations where provided)
- **TLS/TLS7 standard materials** and examples (for structured, layered editions)
- Additional files accompanying the release
